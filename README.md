# ADHD Mode / ADHD 专注模式

> **English** | [中文](#中文)

**ADHD Mode** is a macOS application designed to help you enter a focused work session by dimming distractions and providing subtle audio cues.

**ADHD Mode** 是一个 macOS 应用程序，旨在通过屏蔽干扰和播放舒缓的音频提示，帮助您进入专注的工作状态。

## Features / 功能特性

- **Full Screen Overlay**: Dims the background and hides distractions (Dock, Menu Bar).
  - **全屏覆盖**：压暗背景，隐藏干扰项（如 Dock 栏和菜单栏）。
- **Looping Audio**: Plays a subtle audio cue to keep you in the zone.
  - **循环音频**：播放细微的音频提示，助您保持专注。
- **One-time Session**: Runs for a set duration and then automatically exits.
  - **单次会话**：运行设定时间后自动退出。

## Usage / 使用方法

Simply run the app to enter ADHD Mode.
只需运行 App 即可进入专注模式。
